═══════════════════════════════════════════════════════════════════════
                         PANEL FEATURES
═══════════════════════════════════════════════════════════════════════

✅ Health Status Indicator (Left Bar)
   - 🟢 Green: All systems operational
   - 🟡 Yellow: Warning condition
   - 🔴 Red: Critical/Emergency

✅ Semi-transparent Background
   - Dark glass effect for visibility
   - Doesn't obscure map underneath

✅ Color-Coded Information
   - Cyan: Headers and titles
   - White: Normal data
   - Yellow: Warnings
   - Red: Critical alerts

✅ Battery Visual Indicator
   - Progress bar at bottom
   - Color changes: Green → Yellow → Red

✅ Connection Line
   - Dotted line connects panel to drone
   - Color matches health status

✅ Auto-positioning
   - Automatically positions to avoid screen edges
   - Follows drone as it moves

═══════════════════════════════════════════════════════════════════════
                      REAL-TIME UPDATES
═══════════════════════════════════════════════════════════════════════

All data updates in real-time as the drone moves:

⏱️  Battery Level        → Decreases based on flight dynamics
⏱️  Flight Time         → Recalculated continuously
⏱️  Speed/Acceleration  → Updates with position changes
⏱️  Attitude (P/R/Y)    → Calculated from maneuvers
⏱️  Sensor Status       → Real-time health monitoring
⏱️  Mission Progress    → Waypoint advancement tracking

═══════════════════════════════════════════════════════════════════════
                     DYNAMICS CALCULATION
═══════════════════════════════════════════════════════════════════════

The drone object automatically calculates:

📐 Bearing Change Rate
   - Measures turning speed (deg/s)
   - Updates roll angle based on turn rate

📐 Acceleration
   - Change in velocity over time
   - Updates pitch angle based on acceleration

📐 Climb Rate
   - Vertical speed from altitude changes
   - Affects battery consumption

📐 Battery Drain Model
   Base Drain × Speed Factor × Vertical Factor × Maneuver Factor
   
   Factors:
   - Speed: Higher speed = more drain
   - Vertical: Climbing/descending = more drain
   - Maneuver: Turning/accelerating = more drain

═══════════════════════════════════════════════════════════════════════
                      HOW TO USE
═══════════════════════════════════════════════════════════════════════

1. Run the radar display application
2. Wait for tracks to appear on PPI
3. Right-click any track
4. Select "🎯 Focus Track" from menu
5. Detailed drone panel appears!
6. Panel follows the drone automatically
7. Right-click again to unfocus
